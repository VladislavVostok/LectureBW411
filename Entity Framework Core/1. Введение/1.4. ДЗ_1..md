
# Создание таблиц в Entity Framework Core

**Тема базы данных:** "Фитнес-трекер" - система для учета тренировок, упражнений, клиентов и тренеров.

---

## **Задание**

Создайте простое консольное приложение с использованием Entity Framework Core, которое реализует базу данных для фитнес-трекера. Вам необходимо создать 4 независимые таблицы без связей между ними.

### **Требования:**

1. Создайте 4 сущности (классы) по тематике фитнес-трекера.
2. Настройте эти сущности для создания соответствующих таблиц в базе данных.
3. Реализуйте миграцию для создания базы данных.

---

## **Описание таблиц:**

### **1. Таблица `Exercises` (Упражнения)**
Содержит информацию о различных упражнениях.

| Поле              | Тип    | Описание                                            |
| ----------------- | ------ | --------------------------------------------------- |
| Id                | int    | Первичный ключ                                      |
| Name              | string | Название упражнения                                 |
| DifficultyLevel   | string | Уровень сложности (Начальный, Средний, Продвинутый) |
| EquipmentRequired | string | Необходимое оборудование                            |
| TargetMuscleGroup | string | Целевая группа мышц                                 |

### **2. Таблица `Clients` (Клиенты)**
Содержит информацию о клиентах фитнес-центра.

| Поле             | Тип      | Описание                                    |
| ---------------- | -------- | ------------------------------------------- |
| Id               | int      | Первичный ключ                              |
| FullName         | string   | Полное имя клиента                          |
| BirthDate        | DateTime | Дата рождения                               |
| MembershipType   | string   | Тип абонемента (Разовый, Месячный, Годовой) |
| RegistrationDate | DateTime | Дата регистрации                            |

### **3. Таблица `Trainers` (Тренеры)**
Содержит информацию о тренерах.

| Поле            | Тип    | Описание                              |
| --------------- | ------ | ------------------------------------- |
| Id              | int    | Первичный ключ                        |
| FullName        | string | Полное имя тренера                    |
| Specialization  | string | Специализация (Йога, Силовые, Кардио) |
| ExperienceYears | int    | Стаж работы (лет)                     |
| PhoneNumber     | string | Контактный телефон                    |

### **4. Таблица `WorkoutSessions` (Тренировочные сессии)**
Содержит информацию о проведенных тренировках.

| Поле | Тип | Описание |
|------|-----|----------|
| Id | int | Первичный ключ |
| Date | DateTime | Дата проведения тренировки |
| DurationMinutes | int | Продолжительность в минутах |
| CaloriesBurned | int | Сожженные калории |
| SessionType | string | Тип тренировки (Групповая, Индивидуальная) |

## **Требуемые действия:**

### **Часть 1: Создание проекта и настройка EF Core**
1. Создайте новое консольное приложение .NET
2. Установите необходимые пакеты NuGet:
   - Microsoft.EntityFrameworkCore.Sqlite
   - Microsoft.EntityFrameworkCore.Tools
3. Создайте класс `FitnessTrackerContext`, наследующий от `DbContext`

### **Часть 2: Создание сущностей**
Создайте 4 класса для представления таблиц (Exercises, Clients, Trainers, WorkoutSessions) с соответствующими свойствами.

### **Часть 3: Настройка DbContext**
В классе `FitnessTrackerContext`:
1. Добавьте DbSet для каждой сущности
2. Переопределите метод `OnConfiguring` для настройки подключения к SQL Server
3. Используйте строку подключения: `"Server=(localdb)\\mssqllocaldb;Database=FitnessTrackerDB;Trusted_Connection=True;"`

### **Часть 4: Создание и применение миграции**
1. В конструкторе `FitnessTrackerContext` пропишите строчку `Database.EnsureCreated();`
