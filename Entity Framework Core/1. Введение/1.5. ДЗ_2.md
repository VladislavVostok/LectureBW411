# CRUD-операции для фитнес-трекера в Entity Framework Core

**Тема базы данных:** "Фитнес-трекер" - система для учета тренировок, упражнений, клиентов и тренеров.
## **Задание**

На основе предыдущего задания вы создали `консольное приложение` с использованием Entity Framework Core для создания таблиц. Далее мы усовершенствуем это приложения и добавим пункты меню для управления данными фитнес-центра. Нужно реализовать полноценные CRUD-операции для 4 таблиц.

### **Требования:**

1. Используйте те же 4 сущности из предыдущего задания
2. Реализуйте консольное меню для выбора таблицы и операции
3. Для каждой таблицы реализуйте все CRUD-операции
4. Добавьте поиск по различным критериям


## **Требуемые операции:**

### **Для КАЖДОЙ из 4 таблиц необходимо реализовать:**

#### **1. CREATE (Добавление)**

- Форма ввода всех полей сущности
- Валидация обязательных полей
- Автоматическая генерация Id

#### **2. READ (Чтение/Поиск)**

- **Показать все записи**
- **Поиск по ID**
- **Поиск по критериям** (для каждой таблицы свои):
    - Для `Exercises`: 
	    - Найти упражнение по ID
	    - Найти упражнения по уровню сложности
	    - Найти упражнения по группе мышц
    - Для `Clients`: поиск по типу абонемента
    - Для `Trainers`: поиск по специализации или стажу
    - Для `WorkoutSessions`: поиск по типу тренировки или диапазону дат

#### **3. UPDATE (Обновление)**

- Поиск записи по ID
- Возможность изменить отдельные поля
- Подтверждение перед сохранением изменений

#### **4. DELETE (Удаление)**
- Поиск записи по ID
- Отображение информации об удаляемой записи
- Подтверждение перед удалением

## **Пару примеров:**

```cs
// Поиск по уровню сложности
public List<Exercise> GetExercisesByDifficulty(string difficulty)
{
    return context.Exercises
        .Where(e => e.DifficultyLevel == difficulty)
        .ToList();
}

// Поиск по группе мышц
public List<Exercise> GetExercisesByMuscleGroup(string muscleGroup)
{
    return context.Exercises
        .Where(e => e.TargetMuscleGroup.Contains(muscleGroup))
        .ToList();
}
```
