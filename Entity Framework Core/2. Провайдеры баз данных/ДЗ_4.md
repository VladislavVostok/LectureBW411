# Установка и настройка PostgreSQL и pgAdmin 4

**Тема:** Подготовка инфраструктуры для использования PostgreSQL с Entity Framework Core.

---

## **Цель задания**
Настроить рабочее окружение для разработки приложений с использованием СУБД PostgreSQL. Выполнив это задание, вы подготовите базу для миграции нашего проекта "Фитнес-трекер" с SQLite/MSSQL на профессиональную клиент-серверную СУБД.

---

## **Задание**

### **Задача 1: Установка PostgreSQL**

1.  **Скачайте установщик:**
    *   Перейдите на официальный сайт: [postgresql.org/download/](https://www.postgresql.org/download/)
    *   Выберите вашу операционную систему (Windows).
    *   Скачайте актуальную версию (рекомендуется 16.x или новее).

2.  **Установите PostgreSQL:**
    *   Запустите установщик.
    *   При установке вам будет предложено выбрать компоненты. Убедитесь, что выбраны:
        *   **PostgreSQL Server**
        *   **pgAdmin 4** (инструмент управления)
        *   **Command Line Tools**
    *   **Важно:** Во время установки вас попросят задать пароль для суперпользователя `postgres`. **Запомните или запишите этот пароль!** Он понадобится для подключения.
    *   Порт оставьте по умолчанию: `5432`.

### **Задача 2: Знакомство с pgAdmin 4**

1.  Запустите **pgAdmin 4** (через меню Пуск).
2.  При первом запуске может потребоваться ввести мастер-пароль (используется для защиты сохраненных паролей в pgAdmin).
3.  В левом меню ("Browser") разверните список серверов (`Servers`).
4.  Подключитесь к локальному серверу (PostgreSQL 16), используя пароль, который вы задали при установке.

### **Задача 3: Создание базы данных для проекта**

1.  В pgAdmin нажмите правой кнопкой мыши на **Databases** -> **Create** -> **Database...**
2.  Введите имя базы данных: `FitnessTrackerDB_PG`.
3.  Нажмите **Save**.
4.  Убедитесь, что база данных появилась в списке.

### **Задача 4: Проверка подключения**

1.  Откройте **Query Tool** для созданной базы данных (правой кнопкой на `FitnessTrackerDB_PG` -> **Query Tool**).
2.  Выполните простейший SQL-запрос, чтобы убедиться, что всё работает:
    ```sql
    SELECT version();
    ```
3.  В окне результатов ("Data Output") вы должны увидеть версию установленного сервера PostgreSQL.
