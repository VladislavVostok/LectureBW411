
1. **Описание задачи**: Вам необходимо разработать программу на C#, которая реализует чтение и запись в текстовый файл несколькими потоками одновременно. Используйте механизмы синхронизации для предотвращения ошибок при одновременном доступе к файлу.
    
2. **Требования**:
    - Создайте текстовый файл, в который один поток будет записывать данные, а другой поток будет читать их.
    - Обеспечьте синхронизацию потоков с помощью lock, Mutex, или других механизмов для того, чтобы несколько потоков не пытались одновременно читать или записывать данные в файл.
    - Один поток должен записывать строку текста в файл каждые 200 миллисекунд, а другой поток должен читать содержимое файла каждые 500 миллисекунд и выводить его в консоль.
    - Программа должна корректно завершаться после того, как в файл будет записано 10 строк.
        
3. **Условия завершения**:
    - Программа завершается, когда запись в файл произведена 10 раз, и после этого происходит финальное чтение данных.
        
4. **Дополнительные задания**:
    - Реализуйте аналогичное задание с использованием классов FileStream или StreamWriter/StreamReader.
    - Добавьте возможность записи нескольких потоков в файл одновременно, используя разные участки файла, и организуйте правильную синхронизацию.

**Цель:**

Вы научитесь использовать потоки для одновременной работы с файловой системой и предотвращать ошибки при параллельном доступе к одному и тому же файлу.
