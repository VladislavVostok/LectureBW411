### Запросы

Необходимо создать базу данных Академия (Academy), которая будет содержать информацию о сотрудниках и внутреннем устройстве академии.

Преподаватели, читающие лекции в академии представлены в виде таблицы Преподаватели (Teachers), в которой собрана основная информация, такая как: имя, фамилия, данные о зарплате, а также дата приема на работу. 

Также в базе данных присутствует информация о группах, хранимая в таблице Группы (Groups). Данные об факультетах и кафедрах содержатся в таблицах Факультеты (Faculties) и Кафедры (Departments) соответственно.

### Описание

База данных Академия (Academy) содержит информацию о сотрудниках и внутреннем устройстве академии.

Преподаватели, читающие лекции в академии представлены в виде таблицы Преподаватели (Teachers), в которой собрана основная информация, такая как: имя, фамилия, данные о зарплате, а также дата приема на работу.

Также в базе данных присутствует информация о группах, хранимая в таблице Группы (Groups). Данные об факультетах и кафедрах содержатся в таблицах Факультеты (Faculties) и Кафедры (Departments) соответственно.

---
### Таблицы

Ниже представлено детальное описание структуры каждой таблицы.

1. **Группы (Groups)**
	1. **Идентификатор (Id)**. Уникальный идентификатор группы.
		1. Тип данных — int.
		2. Авто приращение.
		3. Не может содержать null-значения.
		4. Первичный ключ.
	2. **Название (Name)**. Название группы.
		1. Тип данных — nvarchar(10).
		2. Не может содержать null-значения.
		3. Не может быть пустым.
		4. Должно быть уникальным.
	3. **Рейтинг (Rating)**. Рейтинг группы.
		1. Тип данных - int
		2. Не может содержать null-значения.
		3. Должно быть в диапазоне от 0 до 5.
	4. **Курс (Year)**. Курс (год) на котором обучается группа.
		1. Тип данных - int
		2. Не может содержать null-значения.
		3. Должно быть в диапазоне от 1 до 5.

2. **Кафедры (Departments)**
	1. **Идентификатор (Id)**. Уникальный идентификатор кафедры.
		1. Тип данных - int
		2. Авто приращение.
		3. Не может содержать null-значения.
		4. Первичный ключ.
	2. **Финансирование (Financing)**. Фонд финансирования кафедры.
		1. Тип данных - money
		2. Не может содержать null-значения.
		3. Не может быть меньше 0.
		4. Значение по умолчанию - 0.
	3. Название (Name). Название кафедры.
		1. ▷
		2. ▷ Не может содержать null-значения.
		3. ▷ Не может быть пустым.
		4. ▷ Должно быть уникальным.

3. **Факультеты (Faculties)**
	1. **Идентификатор (Id).** Уникальный идентификатор факультета.
		1. Тип данных - int
		2. Авто приращение.
		3. Не может содержать null-значения.
		4. Первичный ключ.
	2. **Название (Name)**. Название факультета.
		1. Тип данных - nvarchar(100)
		2. Не может содержать null-значения.
		3. Не может быть пустым.
		4. Должно быть уникальным.

4. **Преподаватели (Teachers)**
	1. **Идентификатор (Id).** Уникальный идентификатор преподавателя.
		1. Тип данных - int.
		2. Авто приращение.
		3. Не может содержать null-значения.
		4. Первичный ключ.
	2. **Дата трудоустройства (EmploymentDate).** Дата приема преподавателя на работу
		1. Тип данных - date.
		2. Не может содержать null-значения.
		3. Не может быть меньше 01.01.1990.
	3. **Имя (Name).** Имя преподавателя
		1. Тип данных - nvarchar(max)
		2. Не может содержать null-значения.
		3. Не может быть пустым.
	4. Надбавка (Premium). Надбавка преподавателя.
		1. Тип данных - money
		2. Не может содержать null-значения.
		3. Не может быть меньше 0.
		4. Значение по умолчанию - 0.
	5. **Ставка (Salary)**. Ставка преподавателя.
		1. Тип данных - money
		2. Не может содержать null-значения.
		3. Не может быть меньше либо равно 0.
	6. **Фамилия (Surname)**. Фамилия преподавателя.
		1. Тип данных - nvarchar(max)
		2. Не может содержать null-значения.
		3. Не может быть пустым.
