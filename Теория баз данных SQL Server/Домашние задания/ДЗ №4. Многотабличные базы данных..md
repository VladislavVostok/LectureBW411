### Описание

База данных **Академия** (Academy) содержит информацию о сотрудниках, внутреннем устройстве академии и читаемых лекциях. 

Преподаватели, читающие лекции в академии представлены в виде таблицы **Преподаватели** (Teachers), в которой собрана основная информация, такая как: имя, фамилия и данные о зарплате.

Также в базе данных присутствует информация о группах хранимая в таблице **Группы** (Groups). Данные о факультетах и кафедрах содержатся в таблицах **Факультеты** (Faculties) **Кафедры** (Departments) соответственно. 

Помимо этого, база данных хранит информацию, связанную с проводимыми лекциями. Расписание лекций содержится в таблице **Лекции** (Lectures), а описание дисциплин, по которым читаются лекции, в таблице **Дисциплины** (Subjects).

### Таблицы

Ниже представлено детальное описание структуры каждой таблицы. 

1. **Кураторы (Curators)** 
	1. **Идентификатор (Id)**. Уникальный идентификатор куратора. 
		1. Тип данных - int
		2. Авто приращение. 
		3. Не может содержать null-значения. 
		4. Первичный ключ. 
	2. **Имя (Name)**. Имя куратора. 
		1. Тип данных - nvarchar(max)
		2. Не может содержать null-значения. 
		3. Не может быть пустым. 
	3.  **Фамилия (Surname)**. Фамилия куратора. 
		1. Тип данных - nvarchar(max)
		2. Не может содержать null-значения. 
		3. Не может быть пустым. 

2. **Кафедры (Departments)** 
	1. **Идентификатор (Id)**. Уникальный идентификатор кафедры. 
		1. Тип данных - int
		2. Авто приращение. 
		3. Не может содержать null-значения. 
		4. Первичный ключ. 
	2. **Финансирование (Financing)**. Фонд финансирования кафедры. 
		1. Тип данных - money
		2. Не может содержать null-значения. 
		3. Не может быть меньше 0. 
		4. Значение по умолчанию - 0.
	3. **Название (Name)**. Название кафедры. 
		1.  Тип данных - nvarchar(100)
		2. Не может содержать null-значения. 
		3. Не может быть пустым. 
		4. Должно быть уникальным. 
	4. **Идентификатор факультета (FacultyId)**. Факультет, в состав которого входит кафедра. 
		1. Тип данных - int
		2. Не может содержать null-значения. 
		3. Внешний ключ. 
	
3. **Факультеты (Faculties)** 
	1. **Идентификатор (Id).** Уникальный идентификатор факультета. 
		1. Тип данных - int
		2. Авто приращение. 
		3. Не может содержать null-значения. 
		4. Первичный ключ. 
	2. **Финансирование (Financing)**. Фонд финансирования факультета. 
		1. Тип данных - money
		2. Не может содержать null-значения. 
		3. Не может быть меньше 0. 
		4. Значение по умолчанию - 0.
	3. **Название (Name)**. Название факультета. 
		1.  Тип данных - nvarchar(100)
		2. Не может содержать null-значения. 
		3. Не может быть пустым. 
		4. Должно быть уникальным. 
		
4. **Группы (Groups)** 
	1. **Идентификатор (Id)**. Уникальный идентификатор группы. 
		1. Тип данных - int
		2. Авто приращение. 
		3. Не может содержать null-значения. 
		4. Первичный ключ. 
	2. **Название (Name).** Название группы. 
		1.  Тип данных - nvarchar(10).
		2. Не может содержать null-значения. 
		3. Не может быть пустым. 
		4. Должно быть уникальным. 
	3. **Курс (Year)**. Курс (год) на котором обучается группа. 
		1. Тип данных - int
		2. Не может содержать null-значения. 
		3. Должно быть в диапазоне от 1 до 5. 
	4. **Идентификатор кафедры (DepartmentId)**. Кафедра, в состав которой входит группа. 
		1. Тип данных - int
		2. Не может содержать null-значения. 
		3. Внешний ключ. 

5. **Группы и кураторы (GroupsCurators)** 
	1. **Идентификатор (Id).** Уникальный идентификатор группы и куратора. 
		1. Тип данных - int
		2. Авто приращение. 
		3. Не может содержать null-значения.
		4. Первичный ключ. 
	2. **Идентификатор куратора (CuratorId)**. Куратор. 
		1. Тип данных - int
		2. Не может содержать null-значения. 
		3. Внешний ключ. 
	3. **Идентификатор группы (GroupId)**. Группа. 
		1. Тип данных - int
		2. Не может содержать null-значения. 
		3. Внешний ключ.

		3. Не может содержать null-значения. 
		4. Первичный ключ. 
	2. **Идентификатор группы (GroupId)**. Группа.

7. **Лекции (Lectures)**

		2. Авто приращение.
